# 🏥 Hospital Emergency Room Dashboard | Power BI

## Project Overview
This Power BI dashboard provides a comprehensive analysis of Hospital Emergency Room operations. It helps monitor patient flow, waiting times, satisfaction scores, admission rates, and referral patterns to support data-driven decision-making.

## Key Features
- Total Patient Analysis
- Average Wait Time Monitoring
- Patient Satisfaction Tracking
- Admission vs Non-Admission Analysis
- Age Group Distribution
- Gender and Race Analysis
- Department Referral Insights
- Hourly and Daily Patient Trends
- Interactive Filters and Slicers

## Dashboard Pages
### 1. Monthly View
Analyze ER performance for a selected month.

### 2. Consolidated View
View overall hospital performance and trends.

### 3. Patient Details
Explore detailed patient-level information with dynamic filtering.

## Tools Used
- Power BI Desktop
- Power Query
- DAX
- Data Modeling

## Business Impact
- Improves operational visibility
- Helps identify peak patient hours
- Supports resource planning
- Enhances patient experience
- Enables data-driven healthcare decisions
